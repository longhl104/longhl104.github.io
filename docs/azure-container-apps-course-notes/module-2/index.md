## Overview

This module covers Azure Container Registry and best practices for managing container images.

## Learning Objectives

By the end of this module, you will be able to:

- Create and configure Azure Container Registry
- Push and pull container images
- Implement image security and scanning
- Manage registry access and permissions
- Understand image tagging strategies

## Topics Covered

- Azure Container Registry fundamentals
- Creating and configuring registries
- Building and pushing images
- Image security and vulnerability scanning
- Access control and authentication
- Geo-replication and availability

## Prerequisites

- Completion of Module 1
- Docker installed locally (for hands-on exercises)
- Azure CLI or Azure Portal access

## What's Next

After completing this module, you'll move on to Module 3, where you'll learn how to deploy applications to Azure Container Apps.
