## Overview

Rolldom is a turn-based strategy board game developed by LongCode. This section contains documentation related to the Rolldom app, including privacy policies and other legal documents.

## About Rolldom

Rolldom is a strategic dice-based board game where players compete to conquer territories on a map. The game features:

- **Turn-based gameplay**: Roll dice to attack and conquer regions
- **Online multiplayer**: Play with friends or other players using Unity Relay
- **Minimal data collection**: Privacy-focused design with no personal data storage
- **Local gameplay**: All game settings stored locally on your device
